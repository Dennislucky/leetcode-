# 48. 旋转图像 (Rotate Image)

## 题目描述

给定一个 `n × n` 的二维矩阵 `matrix` 表示一个图像。请你将图像顺时针旋转 90 度。必须原地旋转。

**示例：**
```
输入：matrix = [[1,2,3],[4,5,6],[7,8,9]]
输出：[[7,4,1],[8,5,2],[9,6,3]]
```

## 思路分析

### 转置 + 水平翻转 ✅

顺时针旋转 90° = **先转置（行列互换）** + **再水平翻转（每行反转）**

```
原始：       转置后：       水平翻转后：
1 2 3        1 4 7          7 4 1
4 5 6   →    2 5 8    →     8 5 2
7 8 9        3 6 9          9 6 3
```

## 代码实现

```python
class Solution:
    def rotate(self, matrix: list[list[int]]) -> None:
        n = len(matrix)
        
        # 1. 转置（沿主对角线翻转）
        for i in range(n):
            for j in range(i + 1, n):
                matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]
        
        # 2. 水平翻转（每行反转）
        for i in range(n):
            matrix[i].reverse()
```

## 复杂度分析

- **时间复杂度：** O(n²)
- **空间复杂度：** O(1)

## 关键要点

1. **顺时针90° = 转置 + 水平翻转**
2. 逆时针90° = 转置 + 垂直翻转
3. 转置时只遍历上三角（`j > i`），避免重复交换
