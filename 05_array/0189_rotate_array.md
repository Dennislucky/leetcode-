# 189. 轮转数组 (Rotate Array)

## 题目描述

给定一个整数数组 `nums`，将数组中的元素向右轮转 `k` 个位置。

**示例：**
```
输入：nums = [1,2,3,4,5,6,7], k = 3
输出：[5,6,7,1,2,3,4]
```

## 思路分析

### 三次翻转法 ✅

**核心观察：** 右旋 k 位等于把数组后 k 个元素移到前面。

技巧：通过三次翻转可以原地实现：
1. 翻转整个数组：`[7,6,5,4,3,2,1]`
2. 翻转前 k 个：`[5,6,7,4,3,2,1]`
3. 翻转后 n-k 个：`[5,6,7,1,2,3,4]`

**注意：** k 可能大于数组长度，需要先取模 `k %= n`。

## 代码实现

```python
class Solution:
    def rotate(self, nums: list[int], k: int) -> None:
        n = len(nums)
        k %= n  # 处理 k > n 的情况
        
        def reverse(start, end):
            while start < end:
                nums[start], nums[end] = nums[end], nums[start]
                start += 1
                end -= 1
        
        reverse(0, n - 1)      # 翻转整个数组
        reverse(0, k - 1)      # 翻转前 k 个
        reverse(k, n - 1)      # 翻转后 n-k 个
```

## 过程演示

```
nums = [1,2,3,4,5,6,7], k = 3

第一次翻转（整体）：[7,6,5,4,3,2,1]
第二次翻转（前3个）：[5,6,7,4,3,2,1]
第三次翻转（后4个）：[5,6,7,1,2,3,4] ✅
```

## 复杂度分析

- **时间复杂度：** O(n)
- **空间复杂度：** O(1)

## 关键要点

1. **三次翻转**是原地旋转数组的经典技巧
2. 别忘了 `k %= n`
3. 还有其他方法（额外数组、环状替换），但翻转法最简洁
