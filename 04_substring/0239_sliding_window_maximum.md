# 239. 滑动窗口最大值 (Sliding Window Maximum)

## 题目描述

给你一个整数数组 `nums`，有一个大小为 `k` 的滑动窗口从数组的最左侧移动到最右侧。你只可以看到在滑动窗口内的 `k` 个数字。滑动窗口每次只向右移动一位。返回滑动窗口中的最大值。

**示例：**
```
输入：nums = [1,3,-1,-3,5,3,6,7], k = 3
输出：[3,3,5,5,6,7]
```

## 思路分析

### 单调队列（双端队列）✅

**暴力法**每次窗口内扫一遍找最大值，O(nk)。我们需要 O(n) 的方法。

**核心思想：** 维护一个**单调递减的双端队列**，队头始终是当前窗口的最大值。

**单调队列的性质：**
- 队列中存储的是**下标**
- 队列中的元素对应的值是**从大到小**排列的
- 队头的元素是当前窗口的最大值

**算法步骤：**
1. 遍历数组，对每个元素：
   - **移除过期元素：** 如果队头下标不在窗口范围内（`< i - k + 1`），弹出队头
   - **维护单调性：** 从队尾开始，弹出所有比当前元素小的值（它们不可能是未来窗口的最大值）
   - 将当前下标加入队尾
   - 如果窗口已形成（`i >= k - 1`），队头就是当前窗口最大值

**为什么弹出比当前元素小的值是正确的？**
- 如果 `nums[j] <= nums[i]` 且 `j < i`，那么在 `i` 及之后的窗口中，`j` 永远不可能是最大值
- 因为 `i` 比 `j` 更新（更晚过期）且更大

## 代码实现

```python
from collections import deque

class Solution:
    def maxSlidingWindow(self, nums: list[int], k: int) -> list[int]:
        dq = deque()  # 存储下标，对应值单调递减
        result = []
        
        for i in range(len(nums)):
            # 1. 移除过期的队头（不在窗口内）
            if dq and dq[0] < i - k + 1:
                dq.popleft()
            
            # 2. 从队尾移除所有比当前元素小的
            while dq and nums[dq[-1]] <= nums[i]:
                dq.pop()
            
            # 3. 当前下标入队
            dq.append(i)
            
            # 4. 窗口形成后，记录最大值
            if i >= k - 1:
                result.append(nums[dq[0]])
        
        return result
```

## 过程演示

```
nums = [1,3,-1,-3,5,3,6,7], k = 3

i=0: dq=[0]         (值: [1])
i=1: 弹出0(1<3), dq=[1]         (值: [3])
i=2: dq=[1,2]       (值: [3,-1])      → 窗口[1,3,-1], max=3
i=3: dq=[1,2,3]→[1,3] (值: [3,-3])   → 窗口[3,-1,-3], max=3
i=4: 弹出队头1(过期), 弹出3,2(5>)  dq=[4] (值: [5])  → 窗口[-1,-3,5], max=5
i=5: dq=[4,5]       (值: [5,3])       → 窗口[-3,5,3], max=5
i=6: 弹出5,4(6>)    dq=[6] (值: [6]) → 窗口[5,3,6], max=6
i=7: 弹出6(7>)      dq=[7] (值: [7]) → 窗口[3,6,7], max=7

结果：[3,3,5,5,6,7] ✅
```

## 复杂度分析

- **时间复杂度：** O(n)。每个元素最多入队一次、出队一次
- **空间复杂度：** O(k)，队列最多存储 k 个元素

## 关键要点

1. **单调队列**是解决"滑动窗口中的最值"问题的标准工具
2. 队列存**下标**而非值，方便判断是否过期
3. 每个元素最多入队出队各一次，均摊 O(1)
4. 单调递减队列 → 求最大值；单调递增队列 → 求最小值
